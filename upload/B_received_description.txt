1  설정값 지정
LISTEN_PORT=5000, 수신 폴더 received를 설정함.

2  수신 폴더 생성
os.makedirs(RECV_DIR, exist_ok=True)로 업로드 파일 저장 폴더를 자동 생성함.

3  Flask 서버 생성
app = Flask(__name__)으로 B 수신 테스트 서버를 생성함.

4  업로드 엔드포인트 생성
POST /upload 요청으로 A가 보낸 PCAP 파일을 받음.

5  파일 존재 여부 확인
업로드 요청에 file이 없으면 오류 로그를 남기고 400을 반환함.

6  PCAP 파일 저장
수신한 파일을 received/<파일명> 경로에 저장함.

7  파일 크기 확인
저장된 파일의 크기를 확인해 정상 수신 여부를 로그에 기록함.

8  성공 응답 반환
파일 저장이 끝나면 A에게 200 OK와 "ok"를 반환함.

9  B 테스트 서버 실행
app.run(host="0.0.0.0", port=5000)으로 A의 파일을 받을 서버를 실행함.

10  연결 테스트 목적
변환, 정규화, C 전달 없이 A → B 파일 전송만 먼저 검증함.
